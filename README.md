# Week18 課題 - 要件定義スキル

## 概要
PM的思考の基礎として、要件定義スキルを学んだ。ヒアリング・ユーザーストーリー・機能要件/非機能要件の整理・要件定義書の作成・スコープ管理など、システム開発の上流工程を担うスキルを習得した。

## 成果物

### 基本課題：要件定義書（主婦家計簿）
- ターゲットユーザーの課題を整理し、要件定義書を作成した
- ユーザーストーリー10個・機能要件・非機能要件・MoSCoW分析を含む

### 練習課題①：ヒアリングシート
- 架空のクライアント「Pâtisserie Bonheur」を想定し、ヒアリングシートを作成した

### 練習課題②：ユーザーストーリー
- Pâtisserie BonheurのECサイトの主要機能をユーザーストーリーで記述した

### 練習課題③：要件整理
- 曖昧な要求を明確な要件に整理する練習を行った

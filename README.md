# Week20 課題 - ITビジネススキル

## 概要
ITビジネスの基礎を学んだ。アジャイル開発・スクラム・プロジェクト管理・KPI設定・ビジネスモデル・契約見積もり・3C分析・ペルソナの知識を習得し、主婦家計簿プロジェクトに実際に適用した。

## 実装内容

### スプリント計画
- Week22〜24を3つのスプリントに分割
- ユーザーストーリー10個の優先順位付け（MoSCoW分析をもとに）
- GitHub Projectsでカンバンボードを作成（Todo・In Progress・Review・Done）
- 12タスクをカンバンボードに登録

### KPI設定
- DAU（日次アクティブユーザー数）：リリース3ヶ月後に100人
- 節約目標達成率：利用ユーザーの30%以上が目標達成
- レシート撮影利用率：支出登録の50%以上をレシート撮影で登録

### 見積書作成
- 機能別に単価を設定し見積書を作成
- バッファ30%込みで合計792,000円

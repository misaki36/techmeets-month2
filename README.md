## Week19 課題 - 設計スキル

### 概要
Week18で作成した主婦家計簿の要件定義書をもとに、システム設計ドキュメント一式を作成した。画面設計・DB設計・API仕様書・工数見積もり・インフラ構成図を作成し、「何を作るか」から「どう作るか」への落とし込みを行った。

### 実装内容

- 画面設計（Figma）
  - ログイン・ホーム・支出入力・目標設定・収支一覧の5画面のワイヤーフレーム
  - 画面遷移図の作成
  - プロトタイプ（画面遷移の動作確認）
- DB設計書
  - users・expenses・incomes・categories・goals・family_groupsなど7テーブルの定義
  - ER図の作成
- API仕様書
  - 認証・支出・収入・目標・サマリーなど14エンドポイントの仕様を定義
  - リクエスト/レスポンス例・バリデーション・エラーレスポンスを記載
- 工数見積もり
  - 機能別工数表（合計86時間）
  - バッファ30%込みで112時間と算出
  - Week22〜24の3週間スケジュールを策定
- インフラ構成図
  - EC2・RDS・S3・CloudFrontを使った構成を定義
  - 使用サービス一覧を作成
